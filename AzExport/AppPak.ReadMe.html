AppPak ReadMe

AppPak has not been added to this project. Referenced DLL's will now be added to your exe file as a resource. In order for AppPak to work you will need to modify your code.

In your Program Main method you need to add the following code as the first code to be run. This will allow the dll's to be extracted from the resources and loaded.

AppPakCodeGenerator.AppUnPak unpak=new AppPakCodeGenerator.AppUnPak();

Example Main Function

using System;
namespace TestApp
{
    public class MyAppClass
    {
        [STAThread()]
        public MyAppClass()
        {
            using(AppPakCodeGenerator.AppUnPak unpak=new AppPakCodeGenerator.AppUnPak())
            {
                // your code goes here.
            }
        }
    }
}
