Azure Rest Api Information


Get insights about Azure Rest Apis. Explore them to find available features for every version on every datacenters.

This tool uses two datasources and crosses them to provide you more information about what Azure provides : Azure providers rest Api and Azure swagger definitions.
Azure providers information can be retreive through an authenticated call to :
https://management.azure.com/subscriptions/{subscriptionid}/providers?api-version=2016-09-01

Azure Swagger definitions can be found here
Another official tool on same topic : azure github.io
And of course the most known :resources.azure.com
